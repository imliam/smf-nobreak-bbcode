[center][size=12pt][u][b]Compatibility[/b][/u][/size][/center]

The "[i]NoBreak BBCode[/i]" is compatible for only SMF [b]2.0.X[/b]. It will soon be updated for 1.1.X.

[center][size=12pt][u][b]Features[/b][/u][/size][/center]

The NoBreak BBCode modification adds a BBCode button next to the "Marquee" button on the post page. You can use the [nobreak] BBCode to use this tag. The NoBreak BBCode uses a <nobr> SPAN style tag, which forces all data between the tags to stay on a single line, however if you use a linebreak in the middle of the tag, then it forces a break in it. Also, the <wbr> Wrap Break Tag has been included, but not added to the editor. You can use this by the [wbr] tag.

[center][size=12pt][u][b]Installation[/b][/u][/size][/center]

Install this modification through the package manager. It only extracts one image file, and edits two, so nothing should go wrong.
